Christine Huang


Full-Stack Web Developer | Artist



I am a Software Engineer with a background in art education. I enjoy using front-end and back-end technology to develop user-friendly applications. My passion lies in using logical and creative problem-solving skills when building my projects.

I have always been intrigued with creating art out of materials or resources that appear trivial. For example, transforming a sheet of fabric into a stylish outfit or using colored paints to create emotion-provoking artworks.

I view coding with the same fascination. Stringing seemingly simple letters and symbols together to form a functional application is a deeply-rewarding process. It is also humbling to be able to use my passion to develop applications that will help improve people's daily lives.

My Personal Jungle Garden



Description

This application keeps records plant moms' and dads' plant journey by allowing them to track their plant collections. Users are able to sign up for an account and create plant collections, add plants, and logs for each plant. By keeping a log of each plant, plant moms and dads around the world can learn more about their plants and what makes their plant happy!

Technologies Used:

JavaScript, jQuery, cURL Scripts, JSON, HTML, CSS, Sass, Bootstrap, Handlebars, Figma, Express.js, Mongoose.js, MongoDB, Node.js, Heroku
